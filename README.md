# 🔁 Problemas con Estructuras Repetitivas en Python

## 📚 Descripción

Este repositorio contiene ejercicios desarrollados en Python 🐍 enfocados en el uso de estructuras repetitivas como ciclos `for` y `while`.

Se aplican estos conceptos en la solución de problemas reales que requieren iteración, conteo, acumulación y análisis de datos.

---

## 📂 Contenido

### 🏢 Problema 2 - Control de vecinos

Registro de pagos en una unidad residencial, clasificando vecinos al día o morosos y calculando deuda total.

### 📺 Problema 3 - Encuesta televisión

Análisis de hábitos de consumo televisivo, incluyendo horas, preferencias y disposición de pago.

### ⚽ Encuesta deportes

Registro de preferencias deportivas y análisis de datos de los usuarios.

### 🌡️ Monitoreo de temperatura

Seguimiento de temperaturas con análisis de valores ingresados.

### 💼 Problema 4 - Salarios empresa

Cálculo de promedios salariales, clasificación por antigüedad y aplicación de incrementos.
